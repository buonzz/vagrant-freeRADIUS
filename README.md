# FreeRADIUS Experimental Vagrant Box

This is a vagrant box that will simulate a FreeRADIUS installation in Ubuntu 14 Server

## Requirements

* Vagrant 
* VirtualBox

## Installation

Clone the repo

```
git clone https://github.com/buonzz/vagrant-freeRADIUS.git
```

boot up the box

```
cd vagrant-freeRADIUS
vagrant up
```
